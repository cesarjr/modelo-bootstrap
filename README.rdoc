== Twitter Bootstrap instalado no Rails 3.2 manualmente

Este repositório exemplifica a instalação do framework Twitter Bootstrap
manualmente na asset pipeline do Rails sem o auxílio de nenhuma gem.

Este exemplo baseia-se na conversão do exemplo HERO do Twitter Bootstrap.
Veja abaixo o original:
 http://twitter.github.com/bootstrap/examples/hero.html
